Class: Haxroomie

haxroomie~Haxroomie(opt)

Class for spawning the headless chrome browser and getting sessions for the browser instance.

Each session controls a tab in the headless chrome.

After creating the Haxroomie instance it is required to launch the browser with the createBrowser method before requesting sessions with the getSession method.
