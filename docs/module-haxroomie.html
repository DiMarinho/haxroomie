Type Definitions

Message

Messages are used to emit events through the Session. To recieve these messages you must subscribe to them using the Session#subscribe method. All messages must follow a specific format described here.

The Haxroomie messages are almost similar to Flux Standard Action. The difference is that each message MUST contain the sender. This enables better support for private messages. If the message came from Haxroomie, then sender === session.id.

Message objects can be sent with Session#send and Session#broadcast.
